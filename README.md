# PDF to MySQL Data Extractor

Este projeto extrai emails e telefones de um arquivo PDF e insere esses dados em um banco de dados MySQL.

## Pré-requisitos

- Python 3.6+
- MySQL
- Bibliotecas Python: PyPDF2, mysql-connector-python
